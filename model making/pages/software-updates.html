Keep Software Updated

Software updates often include security fixes for known vulnerabilities. Delaying updates leaves you exposed to attacks that target older versions.

Why updates matter

  • Attackers exploit known bugs. Patches close those holes.
  • Operating systems, browsers, and apps are all targets. Keep everything up to date.
  • Ransomware and malware often rely on unpatched systems.

What to update

  • Operating system: Windows, macOS, Linux, iOS, Android. Enable automatic updates if available.
  • Web browser: Chrome, Firefox, Edge, Safari. Browsers are a common attack surface.
  • Apps and plugins: Office, PDF readers, browser extensions. Remove ones you no longer use.
  • Router and firmware: Check the manufacturer's site or app for firmware updates.

Best practices

Turn on automatic updates where you can. If you must update manually, do it soon after notifications appear. Restart when prompted so changes take effect. Back up important data before major OS updates.
